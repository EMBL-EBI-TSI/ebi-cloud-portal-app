Single Sign On

If you don't have an ELIXIR associated account, you need to create one using one of the supported providers (Google, institutional account, etc.).

This involves email verification, and finishes with a "We've got your registration" message from ELIXIR. When you first log in to the EBI Cloud Portal, you will first be asked if you allow your home institution (or Google, etc) to release your data to ELIXIR, where you can choose to allow/disallow the release of your data to the EBI Authentication Gateway.
